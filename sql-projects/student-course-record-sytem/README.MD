# Student Course Record System

This project is a simple command-line application built with Python and SQLite.

## Purpose
The goal of this project is to practice using SQL with Python by storing and managing student course data in a database instead of files.

## Features (Planned / In Progress)
- Add new students and courses
- View all student records
- Delete records
- Store data using SQLite

## Technologies Used
- Python
- SQLite
- SQL (CRUD operations)

## Status
This project is currently in progress and is being developed as part of learning SQL with Python.
